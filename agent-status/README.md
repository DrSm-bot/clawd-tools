# agent-status

CLI tool to display status of all OpenClaw agent workspaces at a glance.

## Features

- **Last Modified**: Shows when each workspace was last touched (recursive scan, skips node_modules/.git)
- **Git Status**: Branch name + clean/dirty indicator with change count
- **Memory Notes**: Last 3 lines from the most recent `memory/YYYY-MM-DD.md` file

## Installation

```bash
npm install
npm run build
npm link  # Makes 'agent-status' available globally
```

## Usage

```bash
agent-status
```
