# memory-cleanup

CLI tool to analyze and archive memory files across OpenClaw agent workspaces.

## Features

- **Daily Log Count**: Number of `YYYY-MM-DD.md` files per workspace
- **Age Analysis**: Files older than 30 days
- **Size Stats**: Total size of each memory folder
- **Date Range**: Oldest to newest file
- **Archive Mode**: Move old files to `memory/archive/` with `--archive`

## Installation

```bash
npm install
npm run build
npm link  # Makes 'memory-cleanup' available globally
```

## Usage

```bash
# Report mode (default) - just shows stats
memory-cleanup

# Archive mode - moves files >30 days to archive/
memory-cleanup --archive
```
